# Tabs

Модуль для создания контента с содержимым разделяемым по вкладкам.
Поддерживает вид вкладок сверху и слева (настраивается отдельно)

---

Module for creating content with tab-delimited content.
Supports top and left tab view (customizable separately)

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **1.0.0**
